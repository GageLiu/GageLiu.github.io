解决啥子问题？

​ 最近项目要开始正式上线了，为了以防万一数据库数据出错了想要恢复，那么就得给数据库做下备份，当然，所有能计算机完成的事情那么就别用人去完成，要不一个人守着时间每天定时去拷贝一份数据库那就太傻了；所以，我们需要实现的就是三点：

  1. 实现备份
  2. 定时备份
  3. 删除备份

咋子解决

1.实现备份

​ 脚本语言有很多种，大家都熟悉的有python和shell，而linux下当然是用shell方便啦，大家可以先去熟悉一下shell编程的基本语法；

(1)建立一个shell文件“autoBackup.sh”

touch autoBackup.sh

(2)进入文件编辑

vim autoBackup.sh

(3)#!/bin/sh”是对shell的声明，说明你所用的是那种类型的shell及其路径所在

引用：http://blog.sina.com.cn/s/blog_6336857901019zyz.html

（#! /bin/sh 是指此脚本使用/bin/sh来解释执行，#!是特殊的表示符，其后面跟的是解释此脚本的shell的路径）如果没有声明，则脚本将在默认的shell中执行，默认shell是由用户所在的系统定义为执行shell脚本的shell.如果脚本被编写为在Kornshell ksh中运行，而默认运行shell脚本的为C shell csh,则脚本在执行过程中很可能失败。所以建议大家就把”#!/bin/sh”当成C 语言的main函数一样，写shell必须有，以使shell程序更严密。

#!/bin/sh

# Database info
DB_USER="qbtlp2"
DB_PASS="P2i0c1c7@)!&)#_qbtlp2"
DB_HOST="120.79.117.159"
TDDB_NAME="qbtlp2"
